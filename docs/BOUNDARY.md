# Boundary

This repository follows the core OMNIA boundary:

    measurement != inference != decision

---

## Meaning

Measurement may produce a structural signal.

Inference may interpret that signal.

Decision may act on that interpretation.

OMNIA-VALIDATION only validates the artifact and the declared measurement boundary.

It does not make the final decision.

---

## Allowed claims

OMNIA-VALIDATION may claim:

- this artifact was generated;
- this test passed;
- this test failed;
- this result is reproducible under the declared conditions;
- this transformation preserved or broke the measured structure;
- this case is structurally fragile under the declared protocol.

---

## Disallowed claims

OMNIA-VALIDATION should not claim by itself:

- this is semantically true;
- this is physically proven;
- this model is conscious;
- this model is safe;
- this system is insecure;
- this cryptographic object is broken;
- this downstream decision is correct.

---

## External decision

The final decision remains external.

This repository can provide evidence.

It cannot replace judgment.
